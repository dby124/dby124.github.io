laravel数据库事务

可以在 DB facade 上使用 transaction 方法来运行数据库事务中的一组操作。如果在事务 Closure 中发生了异常，事务将自动回滚。而如果 Closure 成功执行，事务将自动被提交。也就是说，使用数据库事务，你就不需要在数据库语句执行发生异常时手动回滚或提交。

注意：DB facade 的事务方法也适用于 查询语句构造器 和 Eloquent ORM 的事务。
