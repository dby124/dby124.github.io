Linux环境搭建

发表于 2018-06-27 | 分类于 Linux

之前对Linux的认知一直主要是做硬件开发用，后来接触了Web编程后了解到，由于成本、灵活性等原因，很多服务器采用的Linux系统，觉得既然以后想从事Web后端开发的方向，不会Linux有点说不过去，所以最近试着去学习Linux的知识，那首先就是要搭建一个Linux系统。

下面是在安装Linux系统的一下笔记，希望在以后的学习中，能够快速地完成。

Linux的环境搭建主要分为启动盘制作–>第一启动项设置–>分区–>系统安装–>基础软件安装

U启动盘制作及Linux磁盘空间划分

1. U启动盘制作步骤：

  • 准备一个空的U盘；
  • 下载并安装老毛桃装机版，下载地址;
  • 使用老毛桃软件制作U盘启动盘，详细教程;
  • 写入系统镜像。
    • 在ISO模式下，点击浏览选择下载好的Ubuntu操作系统镜像文件，然后点击一键制作。
    • 在弹出的“写入硬盘镜像”窗口中，在硬盘驱动器一栏选择你的U盘，选择便捷启动–>写入新的硬盘主引导记录（MBR）–>USB-HDD+或USB-ZIP+,其他选项默认，点击格式化，然后点击写入，等待写入完成即可。

此时U盘启动盘制作成功，下面给Linux划分可用内存。

2. 磁盘空间划分

  • 右击此电脑/计算机–>管理–>磁盘管理，弹出如下图界面;

  • 点击选择一尚有足够空闲空间的磁盘，右击压缩卷，弹出如下界面，在 “输入压缩空间量”一栏填写需要压缩的空间大小，建议大于40G。

  • 等待压缩完成后出现如图是未分配空间，至此为止，保持磁盘未分配状态不要在做其他操作，，该压缩出来的未分配磁盘空间我们将用来安装Ubuntu。

接下来需要重启电脑，设置电脑第一启动项，便于盘启动盘安装系统。

第一启动项设置

插入U盘重启电脑，进入BIOS界面选择你的U盘作为第一启动项。

我的电脑是三星电脑，在开机时通过多次按F2键进入设置。不同品牌电脑进入BIOS的方式有所不同，常见方式有，出现开机logo后快速多次敲击F2、F12、DEL、enter或ESC建，有时具体按键会在开机logo的底部文字内显示。如果不清楚自己电脑按哪个键能进入设置目录，可以进入http://www.xiaobaixitong.com/网站查询。

设置第一启动项并启动U盘系统盘步骤：

  • 进入启动项设置目录。进入Boot–>Boot Device priority。
  • 将电脑的第一启动项设置为U盘。通过F5/F6键将USB HDD移动到第一位，即将U盘设置为第一启动项，然后按F10保存，此时就会通过U盘系统盘启动系统。

具体可参考：三星电脑怎样设置u盘第一启动装系统

进入BIOS选择U盘作为第一启动项后按F10保存并退出，重启系统，进入如系统安装界面。

系统安装

安装步骤：

  • 开始安装。选择“中文（简体）”，在点击“安装Ubuntu”；
  • 然后单击“继续”，接下来会出现问你是否要连接网络，可选择“我现在不想连接wifi网络”或者连网输入密码也可以，然后点击“继续”。关于是否要选“安装第三方软件”，根据需要选择，对后面的安装没有特别大的影响（PS：我选了安装）；
  • “安装类型”窗口，选择“其他选项”，“继续”；
  • 之后手动分盘符，找到其中标有“空闲”的盘符，这个盘符就是我们用于安装ubuntu的磁盘空间，选定“空闲”，然后，点击下方的“+”。注意不要碰别的盘符，小心弄得到时候win7不能用了，甚至品牌机自带的隐藏分区也会被破坏；

  • 四次分区，每次都是从“空闲”中分出部分区域；

    • 第一次分区，点击“+”，进行设置：
      • 挂载点：“/”
      • 大小：22000MB（PS：20多个G，10-20G应该都可以）
      • 新分区的类型：主分区
      • 新分区的位置：空间起始位置
      • 用于：EXT4日志文件系统大小
    • 第二次分区：“空闲”处，继续点“+”，如下设置，
      • 挂载点：（不设置）
      • 大小：2048MB（据说1-2G足够，网上认为是物理内存的2倍，如果可用磁盘容量大，可以设置大一点）
      • 新分区的类型：逻辑分区
      • 新分区的位置：空间起始位置
      • 用于：交换空间
    • 第三次分区：“空闲”处，继续点“+”，如下设置，
      • 挂载点：/boot（网上有的说不需要设置这项，但双系统引导时需要）
      • 大小：200MB（笔者暂时设置为200MB）
      • 新分区的类型：逻辑分区
      • 新分区的位置：空间起始位置
      • 用于：EXT4日志文件系统
    • 第四次分区：“空闲”处，继续点“+”，如下设置，
      • 挂载点：/home
      • 大小：（剩余全部空间，剩下显示多少，就多少）
      • 新分区的类型：逻辑分区
      • 新分区的位置：空间起始位置
      • 用于：EXT4日志文件系统

  • 安装启动引导器的设备。采用默认SDA

    注意：默认是sda，如果选择默认，则是ubuntu引导windows7，到时候卸载ubuntu时，可能会麻烦些，如果想用windows7引导ubuntu，请选择你/boot所在的盘符，之后还要在windows下安装引导类的软件，才能启动ubuntu，过程会复杂些。

  • EasyBCD引导Ubuntu

https://jingyan.baidu.com/article/60ccbceb18624464cab197ea.html
