职业相关课程学习笔记

发表于 2018-03-27 | 分类于 职业规划及求职

记学校就业指导课程，主要记录上课中提到的一些核心知识，在此留下学习的痕迹。

职业生涯规划课程笔记

  • 名人名言

    • 《礼记·大学》：“修身 齐家 治国 平天下”——善待自己，善待家人，善待他人。
    • 孟子：“生于忧患，死于安乐”——如何面对挫折
    • 庄子：“吾生而有涯，而知也无涯”——活到老，学到老

    • 文天祥：“人生自古谁无死，留取丹心照汗青”——人生的意义，留下什么？

      “知行合一”——行动力

  • 马斯洛需求层次理论：

    • 生理需求（Physiological needs）
    • 安全需求（Safety needs）
    • 爱和归属感（Love and belonging）
    • 尊重（Esteem）
    • 自我实现（Self-actualization）

    • 自我超越需求（Self-Transcendence needs）（非必要）

      天中小语：生气不如争气。

  • “不知死，焉知生”，生死学中，库伯勒·罗斯说到，人濒死五个阶段：否认期、愤怒期、妥协期、沮丧期和接受期。

    躯体生命可以结束，精神生命是无尽的。

建立自己的职业生涯

  • 生涯的三个阶段：

    • 生存期，现有的能力、业余爱好。
    • 过渡期，兴趣转化、平衡自我。
    • 事业期，内在统和、自我实现。

  • 求职前自我定位的三个步骤：

    • 总结个人经历、实践、成就、证书，梳理个人能力；
    • 梳理人脉和资源，在求职过程中谁可以帮助你？
    • 在能力基础上，进一步思考价值观、兴趣、性格，选择满足需求的职业。

  • 职业生涯的建议：

    • 当一个人形成了自己的信念的时候，就会确定了人生目标。

    • 如何实现双赢，帮助他人：需要（1）知道对方的需求（2）知道自己的需求。

    • “一时的负债，可能会成为将来最好的资产”

    • 冷静 VS. 冷漠：冷漠是坏习惯，冷静是重新思考、面对问题。

    • 感恩的真正原因，是来自于大爱。大爱是一种付出，他永远不求回馈。

    • 休闲活动带来的好处：

      • 缓解学业、工作的压力；
      • 提高学习和工作的效率；
      • 激发创造力。

    • 只有提高自己的核心竞争力，才有立足之地。

      天中小语：没有抱怨。只有面对。

  • 如何面对目前的职业生涯规划

    • 拥抱国家的政策。现在国家政策有：自贸区（上海、广东、天津、福建）、一带一路、大健康产业链、东盟、互联网+等政策。

      世界经历了机械化时代、电气化时代、计算机时代、信息网络时代，当前已到来的是健康保健时代。

    • 洞察市场的需求。

    • 确定胜任愉快的工作。

      最重要的是如何定位最合适自己的职业生涯规划。

  • 共勉

    • 信息化、全球化、未来化

    • 企划力、竞争力和执行力

      天中小语：我心若静，风奈我何？

面试

  • 展示真实的自己，与简历一致。善于发现和挖掘自己的优点。
  • 敢于承认自己的错误。对于未涉及的知识点勇敢地承认。
  • 减小形容词，具体和量化地讲，用证据来证明自己的能力。回答问题也要具体和明确，要成为问题的解决者。

  • 以自在的状态来面试。曾经有个面试者在等待面试的时候非常的从容，在别人焦虑的时候，她自己带了一本小说在看，然而作为第一个面试者，她从而以对。在被问到“如何面对失败？”的时候，表现出来的是一种积极的态度。当场面试官就决定录取她。主要在于其自由的状态和积极的态度。

    天中小语：掌声是给解决问题的人。

财政自由

  • 复利：就是把投资的本金算成利息后，这个利息又折成本金继续计算利息的盈利过程。

    建议，在资金较少的时候，选择高回报的投资。具有理财的观念是极其重要的。

  • 财务自由：不工作，也能承担生计，不要为了养家糊口工作，可以做自己喜欢做的事情。

生活品质和感情账户

  • 感情账户：在有舒适接触之后，产生安全感之后建立的。

    同理心（enpathy）：从对方的角度去思考，理解和倾听，这样才能建立情感。拉近人与人之间的关系。

演讲与口才

卡耐基说：成功 = 15%的专业技能 + 85%的人际关系。

  • 口才：有效的有艺术地表述自己的真实想法。口语表达力有了较高的技巧和艺术水平，具有以下特点

    • 言之有理，不是歪理邪说。
    • 言之有物，不是空洞无物。
    • 言之有序，有条有理，条清理晰。
    • 言之有文，生动形象。
    • 言之有情，感情充沛，真挚真诚。

  • 口才的最高境界

    • 言不在多，达意则灵。
    • 个人智慧的集中表现。
    • 说话内容体现理性美丽和智慧的光辉。

  • 口才需要具备的七要素

    • 德是口才的灵魂
    • 才是口才的核心
    • 学是口才的基础
    • 识是口才的方向
    • 胆是口才的条件
    • 情是口才的保证

    • 体是口才的前提

      能说话 + 会说话 = 说好话。
