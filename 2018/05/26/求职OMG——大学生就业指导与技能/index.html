求职OMG——大学生就业指导与技能

发表于 2018-05-26 | 分类于 职业规划及求职

简历OMG

  • Objective：锁定目标。
    • 求职意向，就是求职的目标（简历中放在简历的前1/3的部分，或者是基本信息的下面，或者是简历的右上角）。
    • 简历的类型：（1）专项简历（针对特定的公司和岗位的简历）;（2）通用简历，在不知道会有哪些应聘公司的时候，采用通用简历。
  • Match：匹配原则。
    • 素材准备，回顾过往人生，可以事先用Excel表格列出。
      • 教育经历：最近的两个，最近的放在前面，如硕士，然后再是本科。
      • 工作经历：实习、课题和项目等。
      • 社团经历
      • 奖励证书
    • 匹配：
      • 招聘简章：简历的GPS，提取招聘简章中关键字，结合素材进行编辑。
      • 职场要求：如果内容太多，那么就需要“断舍离”，优先舍离的顺序是：兴趣爱好，性格评估，专业课程，能力中不重要的部分，经历中不重要的部分。
  • Go for it：努力争取原则，给HR一个完美的阅读体验，遵从的原则：
    • 三三制原则：选3个经历，每个经历用3个短句（不要主语）。
    • PAP原则：用Problem-Action-Result的形式来说明经历，例子和数字来体现。

注意：HR看简历的平均时间为25s，且其注意力呈现的是一个F型形状，所以，重要的部分要放在简历的前1/3的部分。

邮箱投递简历注意事项：

  • 邮箱：用QQ邮箱投递，注意不要使用太过奇怪邮箱昵称，尽量使用自己的名字作为昵称。
  • 要求：根据投递要求，如简历是附在正文还是以附件的形式。发完简历之后给HR打电话或者发短信提醒.

宣讲会，招聘会，面试时，多带几份简历。

怎么向面试官提问

  • 表明自己很适合工作（向技术面试提问）
    • 虚心请教，如：我非常想获得这份工作，您有什么书或者技术推荐我先学习一下吗？您能告诉我这份工作较具挑战性的部分在哪里吗？
    • 入职会进行培训，还是直接工作呢？
    • 什么能够得到面试结果？
    • 什么时候能入职呢？
  • 问一些更大格局的问题（向Boss和HR提问）
    • 对入职的新人，公司需要怎么的要求呢？
    • 什么样的职员在公司有比较好的发展呢？
    • 关注公司的最新动态，针对性的提问。
